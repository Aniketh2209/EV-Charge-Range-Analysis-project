# EV Charge Range Analysis Project

## Overview

The **EV Charge Range Analysis Project** analyzes electric vehicle charging behavior and driving range using data visualization.
The goal of this project is to understand EV performance, charging patterns, and range efficiency through interactive dashboards and data analysis.

Electric vehicles are becoming increasingly important for sustainable transportation. This project helps visualize and analyze EV data to identify useful insights.

---

## Objectives

* Analyze electric vehicle charging patterns
* Study the relationship between battery capacity and driving range
* Visualize EV performance using dashboards
* Identify trends in EV charging infrastructure and usage

---

## Technologies Used

## Dataset

The dataset contains information related to electric vehicles including:

* Vehicle model
* Battery capacity
* Charging time
* Driving range
* Energy consumption

These datasets are stored in the **Datasets** folder of the repository.
